# CoPilot Pour Automatisation de test

Ce dépôt a pour objectif de démontrer l’utilisation de GitHub Copilot avec le framework WebEngine pour l’automatisation de tests. Il contient une solution composée de trois projets :


## Linear Scripting
Ce projet illustre un usage basique de Copilot pour écrire des scripts simples d’automatisation.


## ReqNRoll.Tests

Ce projet permet d’automatiser un parcours de souscription d’un produit d’assurance habitation en utilisant l’approche Gherkin.
Il s’appuie sur Reqnroll (remplaçant de SpecFlow) et sur WebEngine.


## KeywordDriven.Tests

Ce projet propose une automatisation basée sur le Keyword Driven Testing.
Le scénario est découpé en plusieurs mots-clés, avec une externalisation des jeux de données.
